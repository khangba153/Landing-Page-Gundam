# Master Prompt

Sử dụng file này để lưu prompt tổng hợp cho các vòng tiếp theo khi mở rộng landing page.

## Hướng mở rộng đề xuất

- Nâng cấp hero thành key visual đầy đủ với hình ảnh mô hình thực
- Thêm slider cho featured products
- Cải thiện copywriting theo hướng high-conversion
- Tối ưu responsive và hiệu ứng cuộn trang
